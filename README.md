# UcMultiFileSelecTest

控件的目的：多文件选择列表，排序  
用于文件列表有顺序处理的场景（比如顺序合并）  
对于ObjectListView的处理方式可以应用于其它需要排序的场景  
比如，工艺路线的列表，工艺可能会位置调整
